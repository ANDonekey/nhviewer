# 低保真线框图（Lo-Fi）

> 说明：用于确定信息布局与交互优先级，不代表最终视觉设计。

## 1. 首页/热门

`	ext
+------------------------------------------------+
| 顶栏: NHViewer                  [搜索] [筛选]    |
+------------------------------------------------+
| Tab: 热门 | 最新 | 推荐                          |
+------------------------------------------------+
| [封面] 标题                     标签/页数/评分    |
| [封面] 标题                     标签/页数/评分    |
| [封面] 标题                     标签/页数/评分    |
| ... 分页加载 ...                                 |
+------------------------------------------------+
| 首页 | 搜索 | 收藏 | 历史 | 设置                 |
+------------------------------------------------+
`

## 2. 搜索 + 标签筛选

`	ext
+------------------------------------------------+
| <- 返回   [关键词输入框.....................] [X] |
+------------------------------------------------+
| 排序: [热门 v]  时间: [全部 v]  语言: [全部 v]    |
| 标签筛选: [+ 添加标签]  已选: [tag1] [tag2]       |
+------------------------------------------------+
| 结果列表（分页）                                 |
| [封面] 标题                                      |
| [封面] 标题                                      |
+------------------------------------------------+
| [重置筛选]                      [应用筛选]        |
+------------------------------------------------+
`

## 3. 详情 + 预览

`	ext
+------------------------------------------------+
| <- 返回                        [收藏] [下载]      |
+------------------------------------------------+
| [大封面] 标题                                     |
| 作者/语言/页数/分类                               |
| 标签: [tag] [tag] [tag] ...                       |
+------------------------------------------------+
| 缩略图预览网格                                    |
| [1] [2] [3] [4]                                   |
| [5] [6] [7] [8]                                   |
+------------------------------------------------+
| [开始阅读]                    [继续阅读(第xx页)]   |
+------------------------------------------------+
`

## 4. 阅读器

`	ext
+------------------------------------------------+
| <- 返回      标题                  [菜单]         |
+------------------------------------------------+
|                 当前图片页                         |
|         (支持缩放、双击、横竖屏切换)              |
|                                                    |
+------------------------------------------------+
| 上一页      进度: 12/120      下一页               |
+------------------------------------------------+
| 菜单: 画质 | 预加载 | 方向锁定 | 跳页 | 下载       |
+------------------------------------------------+
`

## 5. 下载管理

`	ext
+------------------------------------------------+
| <- 返回                 下载任务                 |
+------------------------------------------------+
| 任务A  35%  [暂停] [取消]                        |
| 任务B  失败  [重试] [删除]                       |
| 任务C  已完成 [打开目录]                          |
+------------------------------------------------+
| 全部暂停 | 全部恢复 | 清理已完成                 |
+------------------------------------------------+
`

## 6. 收藏/历史

`	ext
+------------------------------------------------+
| 收藏                                            |
+------------------------------------------------+
| [封面] 标题        最近阅读: 第 32 页            |
| [封面] 标题        最近阅读: 第 10 页            |
+------------------------------------------------+
| 历史                                            |
+------------------------------------------------+
| [封面] 标题        今天 14:30                    |
| [封面] 标题        昨天 21:10                    |
+------------------------------------------------+
`

## 7. 设置

`	ext
+------------------------------------------------+
| 设置                                            |
+------------------------------------------------+
| 图片质量            [高 v]                       |
| 缓存大小            [500MB] [清理缓存]           |
| 并发下载            [3]                          |
| 主题                [跟随系统 v]                 |
| 语言                [简体中文 v]                 |
+------------------------------------------------+
`
